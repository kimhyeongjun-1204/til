# [프로그래머스] 무인도 여행 - BFS

## 문제 요약
[문제 링크] https://school.programmers.co.kr/learn/courses/30/lessons/154540
- 2D 격자에서 연결된 숫자 칸들을 하나의 섬으로 묶고, 각 섬의 합을 오름차순 반환

## 내가 처음에 생각한 방식 

- DFS 선택 이유 : 끊어지지 않는 숫자 칸들을 연결하면서 값을 합해야 하므로 처음엔 DFS를 선택함. 


## 처음으로 구현한 코드 

```java
    int dfs(int i, int j, int[][] map, boolean[][] visited) {
        int[][] move = {{1,0}, {0,1} , {-1,0}, {0,-1}}; 
        
        for(int k=0;k<4;k++) {
            int nI = i + move[k][0];
            int nJ = j + move[k][1]; 
            
            if(nI >= 0 && nI < R && nJ >=0 && nJ < C) {
                if(map[nI][nJ] != 0 && !visited[nI][nJ]) {
                    visited[nI][nJ] = true;
                    dfs(nI, nJ, map, visited); 
                }
            }
        }
        
        return map[i][j]; 
    }

```

## 해당 코드의 문제 

### 각 연결된 칸의 값을 누적해가야 하는데 해당 칸의 값만 리턴해주고 있음 => 결국 dfs를 진행해도 처음 칸의 값만 리턴됨

<br/>

## 해결 방법 

- sum 변수를 이용하여 dfs를 진행할때마다 각 칸의 값을 누적함 

- 코드 수정 : <br/>

```java
    int dfs(int i, int j, int[][] map, boolean[][] visited) {
        int[][] move = {{1,0}, {0,1} , {-1,0}, {0,-1}}; 
        int sum = map[i][j]; 
        
        for(int k=0;k<4;k++) {
            int nI = i + move[k][0];
            int nJ = j + move[k][1]; 
            
            if(nI >= 0 && nI < R && nJ >=0 && nJ < C) {
                if(map[nI][nJ] != 0 && !visited[nI][nJ]) {
                    visited[nI][nJ] = true;
                    sum += dfs(nI, nJ, map, visited); 
                }
            }
        }
        
        return sum; 
    }

```
<br/>


## 배운 점
- DFS에서 연결된 영역의 값을 합산할 때는 재귀 호출의 반환값을 반드시 누적해야 한다.
  호출만 하고 반환값을 무시하면 현재 칸의 값만 리턴되는 실수가 생긴다.
- 이 패턴은 "섬의 개수" 문제와 다르게, 단순 방문이 아니라 값을 모아야 하는 경우에 주의할 점이다.
